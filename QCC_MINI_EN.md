## What Is QCC

QCC is a blockchain operating system built for trusted ledgers and trusted data. It is not a transfer chain, a notarization chain, or a smart contract platform. Just as iOS provides a unified runtime environment for Apps, QCC provides a unified trusted runtime environment for on-chain business. QCC's core proposition in one sentence: **configure rules on-chain — don't write contracts**.

---

## Technical Foundation

QCC's core structure is the container chain — containers are the fundamental on-chain runtime units, each serving exactly one asset domain, achieving complete isolation of assets, rules, and state. The execution layer is driven by three state machines: balance, rule, and index — system boundaries are clear, behavior is verifiable and auditable. The security layer uses the ML-DSA-87 post-quantum signature standard, with a two-factor key derivation combining mnemonic and payment password — a leaked mnemonic no longer means total loss.

## What QCC Can Do

QCC serves two directions simultaneously: financial infrastructure and trusted data service infrastructure. The financial direction covers stablecoins, asset tokens, decentralized exchange, merchant payments, clearing and settlement, and supply chain finance. The data direction covers entity data on-chain, enterprise audit records, judicial notarization, AI behavior provenance, and developer ecosystem. QCC does not simply move assets or data on-chain — it rebuilds the operational logic of finance and the trusted structure of data into the on-chain world itself.

The Genesis Container is QCC's only zero-height container — the root origin point at the moment the protocol is born. It is the only moment in the entire chain's history where transactions, rules, and entity data converge as one — simultaneously carrying genesis distribution, system rules, participants' messages, and the founder's encrypted legacy. A seat in the Genesis Container is not an economic seat — it is a historical seat. It belongs only to those written in before zero height.
